# Experiment 02: Vehicle Management System

**Status:** ✅ Completed

## 📝 Description
An Object-Oriented program to manage vehicle records. It demonstrates the use of **Constructors** (Default, Parameterized, Copy) and **Encapsulation**.

## ✨ Features
* **3 Constructor Types:** Shows different ways to initialize objects.
* **Method Logic:** Includes calculations for mileage based on fuel type (Petrol vs Electric).
* **Tabular Output:** Displays a formatted summary of the fleet.

## 🚀 How to Run
```bash
javac Vehicle.java MainForVehicle.java
java MainForVehicle
```

## 📂 Files
- `Vehicle.java` - Main Vehicle class with constructors and methods
- `MainForVehicle.java` - Driver class to demonstrate functionality
- `README.md` - This file

## 🎯 Key Concepts
- **Encapsulation:** Private fields with public getters/setters
- **Constructors:** Default, Parameterized, and Copy constructors
- **Methods:** start(), drive(), accelerate(), stop(), calcMileage()
- **Arrays:** Managing a fleet of vehicles
