## 🛠️ Tech Stack

- **Language:** Java (JDK 17+)
- **IDE:** IntelliJ IDEA / Eclipse / VS Code
- **Build:** `javac` (standard Java compiler)

## 🚀 How to run an experiment

Use the commands below from a terminal (PowerShell, Command Prompt, or Bash). Replace `Experiment-01-Calculator` with the experiment folder you want to run.

```bash
git clone https://github.com/SnowKingSandy/ProgrammingInJavaLabExperiments.git
cd ProgrammingInJavaLabExperiments/Experiment-01-Calculator
javac Calculator.java
java Calculator
```

Notes:
- Some experiments may contain multiple source files or packages; compile all `.java` files in the folder with `javac *.java` or use an IDE.
- For JavaFX experiments, ensure JavaFX is on the classpath or run them from an IDE configured for JavaFX.
